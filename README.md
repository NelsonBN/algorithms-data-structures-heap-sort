# Algorithms and Data Structures - Heap Sort


## Characteristics
- Time complexity:
    - Best: Ω(n log(n))
    - Average: Θ(n log(n))
    - Worst: O(n log(n))
- Space complexity: O(1)
- In-place
- Unstable
